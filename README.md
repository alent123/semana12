# -----------------------------------------
# 🚀 Desafíos Principales:
#
# ✅ Challenge 1: Auto-Balance After Insert
# Implementación de inserción con rotaciones automáticas (LL, RR, LR, RL).
#
# ✅ Challenge 2: Left & Right Rotations
# Implementación de rotate_left y rotate_right, manteniendo alturas y orden BST.
#
# ✅ Challenge 3: Delete with Rebalance
# Eliminación de nodos y rebalanceo del árbol usando las rotaciones adecuadas.
#
# ✅ Challenge 4: Height & Balance Checker
# Verifica si el árbol cumple las condiciones de balance AVL.
#
# ✅ Challenge 5: Level-Order with Heights
# Recorrido por niveles mostrando key y height de cada nodo.
#
